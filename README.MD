# Stable Diffusion LoRA — Anime Fine-tuning

## Описание проекта

В данном проекте реализуется fine-tuning модели **Stable Diffusion 1.5** на anime-style датасете с использованием **LoRA**.

Цель проекта — адаптировать базовую text-to-image diffusion модель под генерацию изображений в аниме-стиле без полного переобучения всех весов модели.

## Задача

По текстовому prompt необходимо генерировать изображения в более выраженном anime-style домене.

Основная идея — сравнить генерации **до** и **после** LoRA fine-tuning на одинаковых prompt’ах и seed’ах.

## Подход

Модель обучается в несколько этапов:

### 1. Baseline generation

- генерация изображений с помощью базовой **Stable Diffusion 1.5**;
- сохранение результатов до fine-tuning;
- использование фиксированных prompt’ов и random seed для честного сравнения.

### 2. Dataset preparation

- сбор anime-style изображений;
- формирование датасета из пар `image — caption`;
- генерация captions на основе тегов изображений;
- сохранение данных в формате `metadata.jsonl`.

### 3. LoRA fine-tuning

- обучение LoRA-адаптеров для UNet attention layers;
- заморозка основной модели;
- обучение только небольшого числа дополнительных параметров;
- использование mixed precision для экономии памяти.

### 4. Evaluation

- генерация изображений после fine-tuning;
- сравнение результатов before / after;
- визуальная оценка изменения стиля и качества генерации.

## Особенности реализации

- базовая модель: **Stable Diffusion 1.5**;
- метод дообучения: **LoRA**;
- размер датасета: около **1000 image-caption pairs**;
- разрешение обучения: **512×512**;
- LoRA rank: **16**;
- batch size: **1**;
- gradient accumulation: **4**;
- learning rate: **1e-4**;
- training steps: **1000**;
- mixed precision: **fp16**.

## Итог

LoRA fine-tuning позволяет адаптировать Stable Diffusion под anime-style генерацию значительно дешевле, чем full fine-tuning всей модели.

В результате модель сохраняет способность следовать текстовому prompt, но генерирует изображения в стиле, более близком к обучающему anime-датасету.
